# angola
Angola the Realtime Log Inspector

## Requirements

1. node.js >= 4.2.2
1. npm
1. bower
1. gulp

## Installation

1. `$ git clone https://github.com/angola-rabbit/angola.git`
1. `$ cd angola`
1. `$ npm install`
1. `$ npm start`

access to http://127.0.0.1:3000

## Development

1. `$ npm install --global eslint jasmine`
1. `$ npm test`
